One incredible advantage of using R-Markdown is its ability as a platform to be compatible with tons of languages apart from R such as: Python, SQL, Javascript, CSS, and more. R is a powerful tool, but sometimes the need arises to use a different language like Python, that is faster and contains packages that might not be available in R. Here, we explore how to use Python on R-Markdown and an introduction to Python basics.

Functions are very useful because they save us time that would have otherwise been spent rewriting chunks of code over and over again.

# this function takes in a string as its input and reverses it!
def reverse_string (word):
    result = ""
    for i in word :
         result = i + result
    return result      

# call the function
print("your string reversed looks like: ",reverse_string("the house was made of jellybeans"))
## your string reversed looks like:  snaebyllej fo edam saw esuoh eht
